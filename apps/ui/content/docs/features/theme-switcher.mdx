## Installation

```bash
npx loveui@latest add theme-switcher
```

## Usage

```tsx
import { ThemeSwitcher } from "@/components/theme-switcher"
```

```tsx
<ThemeSwitcher />
```

## Features

- Switch between light, dark and system theme
- Controlled and uncontrolled mode
